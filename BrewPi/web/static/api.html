GET methods

  • /setup/ get an object with lists of vessels, pumps, valves, heaters, coolers, and plumbing
  • /vessel/<id> get info about vessel
  • /pump/<id> get info about pump
  • /valve/<id> get info about valve
  • /heater/<id> get info about heater
  • /cooler/<id> get info about cooler
  • /plumbing/<id> get info about pipe section
  • /step/<id> get info about step
  • /recipe/<id> get info about recipe

POST methods

  • /vessel/ add/update a vessel
